# LA Deeptech Map

Interactive Leaflet map of Los Angeles / South Bay deeptech companies — for planning in-person visits.

## Features

- 14 company markers across El Segundo / Hawthorne / Torrance / Long Beach
- Color-coded confidence: green = verified street, amber = city-only, red = unverified
- Sidebar with search (filters by company, address, what they do, funding)
- Click any marker for address, careers link, and funding metrics
- Multi-select + "Route in Google Maps" generates a multi-stop driving route

## Run

Open `index.html` in any browser — no server needed.
